# Engenharia de Prompt com IA Generativa

## 🧠 Contexto e Objetivos

Este projeto tem como objetivo estudar engenharia de prompt utilizando o NotebookLM como ferramenta de aprendizado ativo.

O tema foi escolhido devido à crescente importância da inteligência artificial no mercado de trabalho e à necessidade de saber interagir de forma eficiente com essas ferramentas.

O principal objetivo é aprender a criar prompts mais claros, específicos e eficientes, melhorando a qualidade das respostas geradas pela IA e desenvolvendo pensamento crítico.

O NotebookLM foi utilizado como ferramenta principal para organizar as fontes e realizar interações com a IA, permitindo testar diferentes variações de prompts e analisar as respostas geradas.

## 🤖 Engenharia de Prompts e Aprendizados

Durante o desenvolvimento do projeto, foram realizados testes com diferentes tipos de prompts para melhorar a qualidade das respostas da IA.

### 🔹 Exemplo 1

Prompt inicial:  
"Explique engenharia de prompt"

Problema:  
Resposta muito genérica

Prompt ajustado:  
"Explique engenharia de prompt com exemplos práticos e aplicações no dia a dia"

Resultado:  
Resposta mais clara, detalhada e útil

---

### 🔹 Exemplo 2

Prompt inicial:  
"Quais são as boas práticas?"

Problema:  
Resposta vaga

Prompt ajustado:  
"Quais são as boas práticas para criar prompts eficientes em IA, com exemplos?"

Resultado:  
Resposta mais completa e aplicável

---

### 🧩 Dificuldades encontradas

- Respostas muito genéricas no início  
- Necessidade de refinar os prompts  
- Ajustar o nível de detalhamento  

### ✔ Aprendizado

Prompts mais específicos geram respostas mais úteis e completas.

## 📘 Miniguia de Estudo

### 📌 Resumo

Engenharia de prompt é a prática de criar instruções claras e estruturadas para obter melhores respostas de modelos de inteligência artificial.

Prompts bem elaborados ajudam a melhorar a qualidade, precisão e utilidade das respostas.

---

### 📖 Glossário

- **Prompt:** instrução dada à IA  
- **IA Generativa:** tecnologia que cria conteúdos como texto  
- **LLM:** modelo de linguagem  
- **Contexto:** informação adicional fornecida no prompt  
- **Few-shot:** uso de exemplos no prompt  

---

### 🔁 Prompts Reutilizáveis

- "Explique [tema] de forma simples e depois aprofunde"  
- "Dê exemplos práticos sobre [tema]"  
- "Quais erros evitar em [tema]?"  
- "Como aplicar [tema] na prática?"
